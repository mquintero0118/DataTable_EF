# AccordionItem

> Accordion component

## Props

| Prop name | Description          | Type    | Values          | Default |
| --------- | -------------------- | ------- | --------------- | ------- |
| id        |                      | string  | -               | ''      |
| open      | State of the item    | boolean | `true`, `false` | false   |
| title     | The accordion header | string  | -               | ''      |

## Slots

| Name    | Description         | Bindings |
| ------- | ------------------- | -------- |
| title   | Slot for the header |          |
| default | Main content        |          |
